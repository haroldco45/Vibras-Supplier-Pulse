# 🚀 Vibras Supplier Pulse

**Inteligencia de Datos para la Gestión de Proveedores en Tiempo Real**

---

## 📋 Descripción
**Vibras Supplier Pulse** es una aplicación web ligera y potente diseñada para centralizar y analizar los indicadores clave de desempeño (KPIs) de proveedores específicos. 

Desarrollada para optimizar la toma de decisiones en **Distrileco**, esta herramienta permite cruzar datos críticos de cuatro fuentes distintas sin necesidad de servidores externos, garantizando la privacidad y rapidez en el procesamiento.

## 💡 Características Principales
* **Análisis 360°:** Visualiza Ventas, Inventario, Cuentas por Pagar (CXP) y Compras en una sola pantalla.
* **Cruce Inteligente:** Vincula facturas de compra con estados de cuenta para discriminar deudas por proveedor.
* **Privacidad Total:** El procesamiento se realiza 100% en el navegador del usuario (Client-Side). Los datos nunca salen de tu equipo.
* **Interfaz Moderna:** Diseño limpio, responsivo y fácil de usar.

## 🛠️ Requisitos de Datos
Para que la magia ocurra, la app requiere la carga de archivos en formato `.xlsx` o `.csv` con las siguientes estructuras (mapeo automático):
1.  **Ventas del Mes:** Reporte de salida de mercancía.
2.  **Inventario:** Stock actual en bodega.
3.  **CXP:** Estado de cuenta consolidado.
4.  **Compras:** Detalle de facturas recibidas de proveedores.

## 🚀 Instalación y Uso
1.  Clona este repositorio o descárgalo como ZIP.
2.  Abre el archivo `index.html` en cualquier navegador moderno.
3.  Carga los archivos correspondientes en el panel izquierdo.
4.  ¡Analiza y negocia con datos reales!
