# 🚀 Vibras Supplier Pulse

**Inteligencia de Datos para la Gestión Estratégica de Proveedores en Tiempo Real**

---

## 📋 Descripción
**Vibras Supplier Pulse** es una solución de *Business Intelligence* (BI) ligera y potente. Está diseñada para centralizar y analizar los indicadores clave de desempeño (KPIs) de proveedores específicos, permitiendo una negociación basada en datos reales y actualizados.

Desarrollada para optimizar la toma de decisiones en **Distrileco**, esta herramienta permite cruzar datos críticos de cuatro fuentes distintas de manera local, garantizando privacidad absoluta y rapidez inmediata.

---

## 💡 Características Principales
* **🔍 Análisis 360°:** Visualiza Ventas, Inventario, Cuentas por Pagar (CXP) y Compras en un solo tablero interactivo.
* **🔗 Cruce Inteligente:** Vincula automáticamente facturas de compra con estados de cuenta para discriminar deudas exactas por proveedor.
* **🛡️ Privacidad Total:** El procesamiento es 100% *Client-Side* (en el navegador). Los datos sensibles de la empresa nunca salen de tu equipo ni se suben a la nube.
* **📊 Interfaz Gerencial:** Diseño limpio, responsivo y optimizado para una lectura rápida de métricas financieras.

---

## 🛠️ Requisitos de Datos
Para un análisis preciso, la app procesa archivos en formato `.xlsx` (Excel) o `.csv` con las siguientes estructuras:

1.  **Ventas del Mes:** Reporte detallado de salida de mercancía.
2.  **Inventario Hoy:** Stock físico y valorizado actual en bodega.
3.  **CXP:** Estado de cuenta consolidado (Cuentas por Pagar).
4.  **Compras:** Detalle histórico de facturas recibidas de proveedores.

---

## 🚀 Instalación y Uso
1.  **Clonar:** Descarga este repositorio como ZIP o usa `git clone`.
2.  **Ejecutar:** Abre el archivo `index.html` en cualquier navegador moderno (Chrome, Edge, Firefox).
3.  **Cargar:** Sube los 4 archivos correspondientes en el panel izquierdo.
4.  **Analizar:** Selecciona un proveedor y obtén el diagnóstico de negociación al instante.
